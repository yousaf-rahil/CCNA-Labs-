# 🌐 CCNA Labs — Cisco Packet Tracer Collection

> A hands-on collection of 28+ CCNA labs built in Cisco Packet Tracer,
> covering everything from basic networking to advanced routing and security.

---

## 📚 Lab Categories

| # | Category | Labs Covered |
|---|----------|-------------|
| 01 | Fundamentals | Basic networks, Routers, Subnetting |
| 02 | VLANs | VLAN setup, InterVLAN routing |
| 03 | Spanning Tree | PortFast, Root Guard, BPDU Guard |
| 04 | Routing | Static Routing, OSPF |
| 05 | DHCP | Basic DHCP, DHCP with VLANs, DHCP Snooping |
| 06 | Security | Port Security, ACLs |
| 07 | NAT | Static NAT, Dynamic NAT/PAT |
| 08 | Network Services | Syslog, NTP, IP Telephony |
| 09 | Troubleshooting | Network Down, Password Recovery |

---

## 🛠️ Requirements
- Cisco Packet Tracer 8.0 or above
- Basic understanding of networking concepts

Handwritten and digital notes are included in the /Notes folder
